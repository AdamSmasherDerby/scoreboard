CRG Scoreboard - Release Notes

  • Version 3.2.0
    • Added Scoreboard Assistant controls similar to Lineup controls. Only allows setting of Jammer and Pivot.
    • Set operator to 'default' if the login dialog box is closed rather than the login button clicked. Allows for keyboard shortcuts to still be used.
    • When in intermission, if a timeout or official review is started, stop the intermission clock. Reset and restart the intermission clock at the end of the timeout/official review
    • Reduced size of Jam Score +- buttons
    • Return the team name for %t1Nt and %t2Nt if the Twitter altername name is not set
    • Do not set team lead status from skater positions.
    • Add option to pad in the sides of the main scoreboard display to allow for odd width projection environments.
    • Scoreboard display requests the browser goes to fullscreen on click anywhere in the scoreboard display.
    • Convert JQuery animations to CSS3 animations
      • ScoreBoard Display
    • Contributors
      • Michael Mitton
      • Andrew "Keyesboard" Keyes (Scoreboard Assistant screen)
  • Version 3.1.0
    • Changed communications from clients to server to a long poll HTTP model to limit network traffic and increase responsiveness
    • Operator alternate name - Useful for displaying team colors to operator
    • Color support to various pages for team names (see color button on team setup tab)
    • Various overlays from TCDG
    • Per jam score totals
    • Main Scoreboard now has increased customizability from operator controls
      • Three different backgrounds (Black to White, White to Black, Flat Black)
      • Classic scoreboard display or Jam Score Display (with TTO and OR displayed as dots on the outside of the Total Score box)
      • Rounded boxes with gradient backgrounds or Flattened squarish boxes (for venues with lighting difficulties)
    • Skaters in team setup screen are shorted by number, not name
